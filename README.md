## CGMU

CPU governor management utility (rust rewrite)

### Build dependencies

`rustup` and [baker](https://github.com/rv178/baker) (optional)

### Installing

```
bake setup
bake
sudo bake install
```

### Uninstalling

```
sudo bake uninstall
```

### Usage

```
sudo cgmu
```
